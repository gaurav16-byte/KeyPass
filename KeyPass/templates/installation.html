INSTALL-ATION

    STEP 1:

    Head over to Python's official website. Download the installer for your respective operating system. Install it and remember to check the ADD TO PATH option.

    STEP 2:

    Open the Command-Prompt in Windows, or Terminal in MacOS or Linux.

    STEP 3:

    Type the following commands as being shown in the terminal windows.

C:\Users\[user]> pip install wmi flask cryptography prettytable

Collecting cryptography...

$

python app.py

    REMEMBER:

    "pip install" method has to be done just once for the first time when you are running the program.

    NOTE:

    The "python app.py" has to be done everytime you want to open KEYPASS. Open Terminal in the folder where your app.py is stored. Keep all the program files together for them to work properly.
